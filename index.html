Qu'est-ce que le DOM

« Le modèle d'objet de document (DOM) du W3C est une interface indépendante de la plate-forme et du langage qui permet aux programmes et aux scripts d'accéder et de mettre à jour dynamiquement le contenu, la structure et le style d'un document. »

La norme DOM du W3C est séparée en 3 parties différentes :
  • Core DOM - modèle standard pour tous les types de documents
  • XML DOM - modèle standard pour les documents XML
  • HTML DOM - modèle standard pour les documents HTML

C'est quoi donc HTML DOM ?

Le DOM HTML est un modèle d'objet standard et une interface de programmation pour HTML. Il définit :

  • Les éléments HTML en tant que objets
  • Les propriétés de tous les éléments HTML
  • Les méthodes pour accéder à tous les éléments HTML
  • Les événements pour tous les éléments HTML

L'arbre des objets HTML DOM

L'interface de programmation DOM

Le DOM HTML est accessible avec JavaScript (et avec d'autres langages de programmation).
Dans le DOM, tous les éléments HTML sont définis comme des objets .
L'interface de programmation correspond aux propriétés et aux méthodes de chaque objet.
Une propriété est une valeur que vous pouvez obtenir ou définir (comme modifier le contenu d'un élément HTML).
Une méthode est une action que vous pouvez effectuer (comme ajouter ou supprimer un élément HTML).

L'objet "document" du HTML DOM

L'objet document représente votre page Web. Si vous souhaitez accéder à un élément d'une page HTML, vous commencez toujours par accéder à l'objet document.

Voici quelques exemples de la manière dont vous pouvez utiliser l'objet document pour accéder au HTML et le manipuler.

Method Description
document.getElementById(id) Recherche un élément grâce à son id
document.getElementsByTagName(name) Recherche un élément grâce à son nom
document.getElementsByClassName(name) Recherche un élément grâce au nom de sa classe

Property Description
element.innerHTML = new html content Change le contenu HTML interne d'un élément
element.attribute = new value Change la valeur de l'attribut d'un élément HTML
element.style.property = new style Change le style d'un élément HTML
Method Description
element.setAttribute(attribute, value) Change la valeur de l'attribut d'un élément HTML

D'autres opérations sont possibles comme la création d'éléments, la suppression, etc.

Validation de Formulaire en JavaScript

La validation de formulaire en JavaScript permet de vérifier que les champs du formulaire contiennent les données attendues avant l'envoi au serveur. Cela améliore l'expérience utilisateur et évite des requêtes inutiles au serveur en cas de données incorrectes.

Une des méthodes les plus courantes pour valider un formulaire est d'utiliser la méthode onsubmit sur le formulaire pour exécuter une fonction JavaScript avant l'envoi des données.

Exemple

  • Recherche d'un formulaire avec document.forms
  • Validation d'un champ vide
  • Prévention de l'envoi du formulaire avec return false

Modifier le CSS en JavaScript

Le DOM HTML permet à JavaScript de modifier le style des éléments HTML à l'aide de la propriété style. En utilisant cette méthode, vous pouvez modifier diverses propriétés CSS comme la couleur, la visibilité, ou encore la taille des éléments.

Pour modifier le style d'un élément HTML, vous utilisez cette syntaxe : 

document.getElementById(id).style.property = "new value";

Par exemple, pour changer la couleur du texte d'un élément <p> : 

document.getElementById("p2").style.color = "blue";

Utilisation des événements

Vous pouvez également déclencher des changements de style en réponse à des événements, tels que le clic sur un bouton ou le survol d'un élément.

Création d'animations avec JavaScript

Pour créer des animations en HTML avec JavaScript, nous devons utiliser un élément conteneur et un élément animé, puis programmer des changements progressifs dans le style de l'élément animé.

Voici un exemple de structure de base pour une animation :

<div id="container">
  <div id="animate">My animation will go here</div>
</div>

Le code JavaScript

L'animation est réalisée en programmant des changements progressifs dans le style d'un élément. Vous pouvez utiliser un intervalle de temps pour rendre ces changements continus.

id = setInterval(frame, 5); function frame() { if (/* test d'arrêt */) { clearInterval(id); } else { /* code pour changer le style de l'élément */ } }

Exploration des Nœuds DOM avec JavaScript

En HTML, chaque élément est représenté sous forme de nœud dans l'arbre DOM. Voici un exemple basique de la structure DOM d'une page web simple :

<html> <head> <title>DOM Example</title> </head> <body> <h1>Bienvenue dans le DOM</h1> <p>Ceci est un paragraphe.</p> </body> </html>

Code JavaScript pour Naviguer dans le DOM

En utilisant JavaScript, vous pouvez naviguer entre les nœuds, les enfants, les parents, et les frères/sœurs d'une manière dynamique. Voici un exemple de code pour accéder aux nœuds enfants :

let parentElement = document.getElementById("parent"); let childNodes = parentElement.childNodes;
